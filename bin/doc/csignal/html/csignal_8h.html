Detailed Description

This is the main include file for the csignal library. This is the only header file that needs to be included by external libraries/ binaries.

Note
In this library all signals are stored as FLOAT64 values (double) as are all filter values as well as functions performed on signals (e.g. FFT). It is the responsibility of the caller to convert the signals into the format that they require.
Author
Brent Carrara

csignal_error_code csignal_modulate_symbol ( UINT32  in_symbol,
UINT32  in_constellation_size,
FLOAT64 *  out_symbol_inphase,
FLOAT64 *  out_symbol_quadrature 
)

Modulates a data symbol into inphase and quadrature components. Let the data symbol value be m, the constellation size be M. This function returns the following:

signal_inphase = cos( 2 * pi * m' / M ) signal_outphase = sin( 2 * pi * m' / M )

Note
The symbol (in_symbol), m, is first converted to a Gray Code, m', before it is modulated.
Parameters
in_symbolThe symbol to be modulated, m, it must be < than in_constellation_size
in_constellation_sizeThe maximum number of possible symbols. Must be > 0
out_symbol_inphaseThe inphase value returned to the caller if no error is detected.
out_symbol_quadratureThe quadrature value returned to the caller if no error is detected.
Returns
Returns NO_ERROR upon succesful exection or one of these errors: 
    CPC_ERROR_CODE_NULL_POINTER If any parameters are null.
    CPC_ERROR_CODE_INVALID_PARAMETER
      If in_symbol >= in_constellation_size

csignal_error_code csignal_sum_signal ( USIZE  in_signal_length,
FLOAT64 *  in_signal,
FLOAT64  in_scalar,
FLOAT64 *  out_sum 
)

Performs a scalar multiplication of all elements in in_signal by the factor in_scalar and sums the scaled components of in_signal.

Note
This function uses a correlator demodulator to determine if the signal corresponds to the symbols [ -1, 1 ]. The sufficient statistics are calculated as r_i = sum( in_signal * i ), where i is either -1 or 1. The largest r_i is chosen as the correct decision and i is returned.
Parameters
in_signal_lengthThe number of elements in in_signal.
in_signalThe signal to be scaled and summed.
in_scalarThe multiplicative scalar factor to apply to all elements of in_signal.
out_sumThe scaled sum of in_signal.
Returns
Returns NO_ERROR upon succesful exection or one of these errors (see cpc_safe_malloc for other possible errors):

CPC_ERROR_CODE_NULL_POINTER If any of the input parameters are null.
