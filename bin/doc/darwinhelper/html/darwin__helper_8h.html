CHAR* darwin_convert_cfstring_to_char_string ( CFStringRef  in_string_to_convert)

Helper function that creates a new c-style character string by converting in_string_to_convert to an ASCII formatted string. Note that the caller must free the return CHAR*.

Parameters
in_string_to_convertThe CoreFoundation string to convert to a c-style string. The caller must free the returned string.
Returns
A newly created c string that must be freed by the caller.
