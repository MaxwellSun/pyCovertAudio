CHAR * cahal_convert_audio_format_id_to_cstring ( cahal_audio_format_id  in_audio_format_id)

Converts in_audio_format_id to its string representation and returns the newly created string. The caller does not need to free the returned string as the strings are defined constants.

Parameters

Returns
A newly created string representation of the input audio format id.
