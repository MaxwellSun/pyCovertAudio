void cahal_free_device_list ( void  )

Frees the list of cahal_devices. This function will free all devices in the list as well as all members of those devices.

Note
This function will free the global device list that is generated the first time you call cahal_get_device_list(). You should however never need to call this function directly as the global device list is freed when cahal_terminate is called.

CPC_BOOL cahal_test_device_direction_support ( cahal_device *  in_device,
cahal_device_stream_direction  in_direction 
)

Tests to see if in_device is capable of supporting input or output depending on the value of in_direction. Input being recording (e.g. a mic) and output being playback (e.g. a speaker). This function will look for a stream in the in_device struct that supports the passed in direction.

Parameters
in_deviceThe device to test to see if it supports in_direction.
in_directionEither input if in_device is being tested for recording support or output if in_device is being tested for playback support.
Returns
True iff in_device supports audio communication in the direction of in_direction, false otherwise.
