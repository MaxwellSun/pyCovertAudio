Detailed Description

This interface takes care of creating the platform-agnostic device structs as well as the implementation for the audio hardware controls, i.e. playback and record.

AudioObjectPropertySelector  in_property,
FLOAT64 *  out_device_property 
)

Extract the FLOAT64 value of in_property from the audio hardware in_device_id. The extracted value is stored in out_device_property. This is a helper function to wrap the OSX interface for audio hardware.

Parameters
in_device_idThe device handle whose property is to be extracted.
in_propertyThe property to query on in_device_id
out_device_propertyThe location to store the extracted property value.
Returns
Either noErr or an error code. The error code can be passed to darwin_helper for printing.

AudioObjectPropertySelector  in_property,
UINT32 *  out_device_property_value_size,

Generic function to extract both the size and value for in_property from the audio hardware in_device_id. The extracted size is stored in out_device_property_value_size and value is stored in out_device_property_value. This is a helper function to wrap the OSX interface for audio hardware.

Parameters
in_device_idThe device handle whose property is to be extracted.
in_propertyThe property to query on in_device_id
out_device_property_value_sizeThe location to store the number of bytes required to store the property value.
out_device_property_valueThe location to store the extracted property value. The size of out_device_property_value is out_device_property_value_size. The caller must free this parameter.
Returns
Either noErr or an error code. The error code can be passed to darwin_helper for printing.

OSStatus osx_get_device_supported_sample_rates ( cahal_device *  io_device)

Populate the list of supported sample rates in io_device. The caller must free the list of supported sample rates. The list is null-terminated. This is a helper function to wrap the OSX interface for audio hardware.

Parameters
io_deviceThe sample rates supported by the audio device pointed to by the handle in io_device will be logged in the list of sample rates in io_device. This is a null terminated list and must be freed by the caller.
Returns
Either noErr or an error code. The error code can be passed to darwin_helper for printing.
