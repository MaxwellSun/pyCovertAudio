cahal_device_stream *  out_stream

Mallocs a new format description struct and populates it with the the necessary parameters. One block is allocated per supported format.

Parameters
in_bits_per_sampleThe bit depth of the supported format.
in_num_channelsThe number of channels supported in this format.
in_sample_rateThe sample rate supported in this format.
in_num_supported_formatsThe number of supported formats currently in out_stream.
out_streamThe cahal_device_stream struct that holds the null terminated list of supported formats.
Returns
Returns NO_ERROR upon succesful execution (see cpc_safe_malloc for other possible errors).
