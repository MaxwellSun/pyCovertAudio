UINT32 darwin_convert_cahal_audio_format_flags_to_core_audio_format_flags ( cahal_audio_format_flag  in_format_flags)

Converts CAHAL format flags to platform specific flags. Note that the platform agnostic CAHAL flags do form a bitmask of supported flags and therefore are orthogonal to one another.

Parameters
in_format_flagsThe format flags to convert to core audio format flags. These are platform-specific flags.
Returns
The core audio format flag representation of in_format_flag.
