void cahal_initialize (void)
 Entry point for the library. This must be the first function that external libraries call. If not, the system will not be appropriately initialized and the behaviour of the library is undefined.
 
void cahal_terminate (void)
 Termination point for the library. By calling this function all associated OS resources that are held by the library will be returned to the OS. The library is unusable after a call to terminate is performed.

Instantiates a list of audio input/output devices in a platform- agnostic way.

Returns
A list of devices that can be used to interact with the OS' audio hardware. The device list is platform agnostic and must be freed by the caller.
