BOOL ios_get_input_device_info ( cahal_device *  out_input_device)

Configures the only built in input device supported by iOS. This sets the minimum required fields (e.g. handle, device name, sample rate, and number of channels).

Note
CAHAL currently does not support USB microphones or speakers on the iOS platform.
Parameters
out_input_deviceThe input device to configure.
Returns
True iff the minimum fields are set, false otherwise. Note that if False is returned the device is unusable.
