darwin_convert_core_audio_format_id_to_cahal_audio_format_id
cahal_audio_format_id darwin_convert_core_audio_format_id_to_cahal_audio_format_id(UINT32 core_audio_format_id)
This function bridges the core audio format constants to the cahal format constants. It takes as input a core audio format id and returns a cahal format id. The platform specific format valules are not bitmasks and must be checked for equality since only one value can be set at a time.
