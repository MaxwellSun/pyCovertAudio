Enumeration Type Documentation

enum cpc_error_codes

The error codes used by the logger.

Enumerator
CPC_ERROR_CODE_NO_ERROR 

Used to indicate no error.

CPC_ERROR_CODE_NULL_POINTER 

Code used when an operation is beign attempted on a null pointer

CPC_ERROR_CODE_INVALID_LOG_LEVEL 

Used by the logger to indicate an unsupported log level has been used in a log function.

CPC_ERROR_CODE_MALLOC 

Used by cpc_safe_malloc to indicate that the system call to malloc has failed.

CPC_ERROR_CODE_MEMSET 

Used by cpc_safe_malloc to indidate that the system call to memset has failed.

CPC_ERROR_CODE_INVALID_PERMISSIONS 

Used when an operation cannot be performed because the correct permissions have not been obtained.

CPC_ERROR_CODE_INVALID_PARAMETER 

Used when a parameter passed to a function is invalid.

CPC_ERROR_CODE_API_ERROR 

Used when a native OS API is called and it returns an error.
