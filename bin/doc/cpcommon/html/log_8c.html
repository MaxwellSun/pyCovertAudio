Logging function that will either not output the log (if in_log_level is below the current log level) or output the log to either stdout or stderr (if in_log_level is the ERROR level).

Parameters
in_log_levelThe log level the message should be logged at
in_fileThe file name cpc_log was called from
in_line_numberThe line number cpc_log was called on
in_log_formatThe printf formatted format string
...The variable length argument list to use in formatting the log.
Returns
Either CPC_ERROR_CODE_NO_ERROR if no error is detected or an error.
