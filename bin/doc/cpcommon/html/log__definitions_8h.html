  CPC_LOG_LEVEL_TRACE = 0, CPC_LOG_LEVEL_DEBUG, CPC_LOG_LEVEL_INFO, CPC_LOG_LEVEL_WARN,
  CPC_LOG_LEVEL_ERROR, CPC_LOG_LEVEL_NO_LOGGING
}
 The supported log levels. Trace is the lowest (0), error is the highest (4). When the log level is set only CPC_LOG calls with a log level equal to or above the current log level will be printed. NO_LOGGING is used to disable logging (usually for tests).
